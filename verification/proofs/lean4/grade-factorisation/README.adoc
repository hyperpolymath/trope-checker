= grade-factorisation — tropical factorisation probe

A dynamics-free, purely algebraic probe of the trope-particularity grade. It asks
whether the grade admits a homomorphic projection onto a *commutative tropical cost*,
with all non-commutativity confined to the complement.

It *imports* the proved carrier from the sibling `grade-boundary/` (it does not
redefine or modify it) and pins the same toolchain (Lean 4 v4.16.0 + Mathlib v4.16.0),
sharing that build through a symlinked `.lake/packages`.

== Verdict

*(B) FACTORS but VERIDICALITY-BLIND.* A nontrivial monoid homomorphism `π : Grade → Δ`
onto the commutative fidelity carrier exists, but *every* such map is forced to identify
the honest `dropped` with the deceptive `falsified`. Tropical captures the *fidelity*
sub-grade only; specifically the *fate-axis* honest/deceptive distinction needs separate,
non-tropical handling, while the commutative `Bond`/`Merge` deceptions stay tropically
separable. The blindness is co-located exactly with the non-commutativity. See
`REPORT.adoc`.

This is a *precondition* result. It does *not* claim recursion is solved, handled, or
addressed; that stays parked behind an operational model (CO-1). See the fireline in
`REPORT.adoc`.

== Run

[source,sh]
----
just build      # green
just falsify    # sorry-free, axiom-clean, falsifier witnesses live
----

== Scope (out of bounds, by design)

No operational model, no fix dynamics, no v0.2. CO-1, O4, the limit carrier,
admissibility, Scott-continuity and `fix`/`lfp` are downstream and untouched.
