== Modelling notes (faithful-but-simplified, recorded for honesty)

* *Provenance tags on `Fused` are erased* in the verified core (they are metadata
  that does not affect the lattice laws); the executable checker carries them.
* *Deceptive duals are modelled as each coordinate's bottom* (worst retention,
  absorbing), which is how the calculus orders `Conflated`; this makes every
  honest floor fail against a deceptive grade (HC-2) by construction.
* *L2 is stated with Œµ's literal value* in Idris (a bare top-level `epsilon` as a
  function argument does not reduce under the unifier); since
  `epsilon = MkGrade Present Present Present Present Intact Single` definitionally,
  the statement is `Œµ ‚ñ∑ g = g`.
* *The use-model floor is a partial per-coordinate demand, not a grade value.* HC-4
  (the Œ£-coherence: bond‚ä£bearer-fate) is enforced on edge grade VALUES, where it
  makes incoherent states unrepresentable; the floor is a set of independent
  minimum-demands (each coordinate optional), so an author CAN write an incoherent
  floor (e.g. bearer-Dropped + bond-Intact). This is harmless ‚Äî each demanded
  coordinate is checked independently ‚Äî but is an author-error surface, not covered
  by the HC-4 unrepresentability claim. (Audit 2026-06-22.)

== Soundness advance (2026-06-22)

Newly machine-checked in `Trope.Order` + `Trope.Soundness` (no axioms):

* *‚äë is a preorder* ‚Äî reflexivity (Trope.Coords) + *transitivity* (`fateLteTrans`,

* *Verdict-soundness corollary* (`verdictSound`) ‚Äî `floor ‚äë g ‚àß g ‚äë œÅ ‚üπ floor ‚äë œÅ`.
  This is the calculus ¬ß8 corollary, now proved (it is exactly transitivity of ‚äë).

  `g ‚äë œÅ`, via congruence of ‚ñ∑ + reflexivity. No L4 needed.
* *L4 monotonicity holds for the chain coordinates* bond and merge
  (`bondComposeMono`, `mergeComposeMono`).

=== FINDING: strict L4 monotonicity FAILS for the fate coordinate

`Trope.Soundness.fateL4MonotonicityFails` machine-checks a counterexample:
`Dropped ‚äë Present` and `Predicated ‚äë Predicated`, yet
`Dropped ‚ñ∑ Predicated = Dropped` while `Present ‚ñ∑ Predicated = Predicated`, and
`Dropped ‚ã¢ Predicated` (Predicated and Dropped are *intentionally incomparable*,
calculus ¬ß3.1). So strict L4 does not hold for fate; the calculus's "L4 is a *lax*
compatibility" (¬ß5 note) is load-bearing. Consequence: grade soundness for
*conservative* (inexact) steps cannot be composed by naive L4 over the fate
coordinate; the *exact* route (`soundFromExact`) sidesteps it, and bond/merge remain
monotone.
